# Issues 目录

存放 agate 项目发现的问题清单。

## 命名规范

`NNN-{slug}.md` — 三位数字编号 + 简短描述。

## 模板

每个 issue 含：
- 日期、严重度、发现场景
- 问题描述（影响 + 实测 + 根本原因）
- 修复方向（方案 A/B/C 对比）
- 状态（待修复 / 待设计 / 进行中 / 已修复）
